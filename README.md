# 欢迎来到我的博客

## 📖 关于本站

这是一个使用 **docsify** 构建的简洁优雅的博客网站。

## 🚀 主要特性

- **多标签功能** - 使用 `docsify-tabs` 插件实现丰富的标签页展示
- **全文搜索** - 快速找到你需要的内容
- **代码高亮** - 支持多种编程语言的语法高亮
- **代码复制** - 一键复制代码片段
- **分页导航** - 便捷的上一章/下一章导航
- **响应式设计** - 完美适配各种设备

## 📚 快速开始

查看侧边栏导航，探索以下内容：

- [技术分享](tech.md) - 技术文章和教程
- [多标签示例](tabs-demo.md) - 多标签功能演示
- [关于我](about.md) - 了解更多关于我的信息

## 💡 提示

点击右上角的菜单按钮可以打开/关闭侧边栏！
