## 快速入门 docsify

### 什么是 docsify？

docsify 是一个轻量级的文档网站生成工具，它不会生成静态的 HTML 文件，而是在运行时动态加载和渲染 Markdown 文件。

### 核心优势

- **无需构建** - 直接使用 Markdown 编写文档
- **简洁优雅** - 默认主题美观大方
- **扩展性强** - 丰富的插件生态
- **SEO 友好** - 支持服务端渲染

### 安装步骤

<!-- tabs:start -->

#### **全局安装**

```bash
npm install docsify-cli -g
```

#### **初始化项目**

```bash
docsify init ./docs
```

#### **启动服务**

```bash
docsify serve docs
```


## 多标签使用技巧

### 基本语法

```markdown
<!-- tabs:start -->

#### **标签名称**

标签内容

<!-- tabs:end -->
```

### 实用场景

1. **多语言代码示例** - 展示不同编程语言的实现
2. **安装选项** - 提供多种安装方式
3. **版本对比** - 展示不同版本的差异
4. **分步教程** - 将教程拆分为多个步骤

### 高级用法

<!-- tabs:start -->

#### **嵌套标签**

可以在标签内再嵌套标签，创建更复杂的内容结构。

#### **持久化状态**

设置 `persist: true` 可以记住用户的标签选择。

#### **同步标签**

设置 `sync: true` 可以让相同名称的标签同步切换。
